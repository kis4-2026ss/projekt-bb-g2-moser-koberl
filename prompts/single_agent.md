# Zero-Shot Prompt: Single-Agent System

Du bist ein erfahrener Full-Stack Entwickler. Deine Aufgabe ist es, eine vollständige E-Commerce Web-Applikation eigenständig zu entwerfen und zu implementieren.

## Technologische Vorgaben:
- **Backend:** Python mit FastAPI
- **Datenbank:** SQLite mit SQLAlchemy als ORM
- **Frontend:** Modulare Web Components (Vanilla JavaScript, Light DOM) mit Tailwind CSS (via CDN)

## Funktionale Anforderungen:
1. **Produktübersicht:** Eine Seite, die alle verfügbaren Produkte mit Bild-Platzhalter, Name und Preis auflistet.
2. **Produktdetailseite:** Eine Ansicht, die beim Klick auf ein Produkt detaillierte Informationen anzeigt.
3. **Warenkorb:** Eine Funktionalität, um Produkte hinzuzufügen, die Liste der gewählten Artikel zu sehen und die Gesamtsumme zu berechnen.
4. **Initialisierung:** Die Applikation muss beim ersten Start automatisch eine SQLite-Datenbank erstellen und mit mindestens 5 Beispiel-Produkten befüllen.

## Erwartetes Ergebnis:
- Erstelle eine saubere Ordnerstruktur.
- Liefere den vollständigen, lauffähigen Code für alle Komponenten (Backend-Logik, Datenbank-Modelle, Frontend-HTML/JS).
- Sorge dafür, dass die Kommunikation zwischen den Web Components und den FastAPI-Endpunkten fehlerfrei funktioniert.
- Der Code muss ohne zusätzliche Konfiguration (außer der Installation der Python-Pakete) startbereit sein.
